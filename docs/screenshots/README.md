# Screenshots

This directory contains screenshots of the NariSetu platform.

## How to Add Screenshots

1. Open the deployed application at https://narisetu-ten.vercel.app
2. Navigate to each section and take a screenshot
3. Save screenshots with the following names:

| Filename | Section |
|---|---|
| `landing-page.png` | Public landing page |
| `dashboard.png` | Intelligence Dashboard |
| `login.png` | Login / Register page |
| `marketplace.png` | Marketplace browse view |
| `ai-coach.png` | AI Business Coach |
| `learning-hub.png` | Learning & Growth Hub |
| `voice-ledger.png` | Voice Ledger |
| `profile.png` | User Profile page |
| `scheme-match.png` | Government scheme results |
| `data-dashboard.png` | Data Dashboard analytics |

## Recommended Screenshot Settings

- **Resolution:** 1920×1080 or 2560×1440
- **Format:** PNG
- **Browser:** Chrome or Edge (for clean rendering)
- **Theme:** Use the default light theme
