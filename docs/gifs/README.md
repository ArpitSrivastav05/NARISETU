# Demo GIFs

This directory will contain animated GIF demonstrations of key user flows.

## How to Record GIFs

Use a tool like [ScreenToGif](https://www.screentogif.com/) or [LICEcap](https://www.cockos.com/licecap/) to record the following flows:

| Filename | Flow |
|---|---|
| `login.gif` | Email login + Google OAuth |
| `scheme-match.gif` | Filling form → receiving matched schemes |
| `ai-coach.gif` | Sending a message → AI response |
| `marketplace.gif` | Browse, filter, contact seller |
| `voice-ledger.gif` | Recording voice → transaction created |
| `learning-hub.gif` | Opening course → taking quiz |
| `product-creation.gif` | Registering business → adding product |
| `intelligence-feed.gif` | Intelligence dashboard loading |

## Recommended GIF Settings

- **Resolution:** 1280×720
- **FPS:** 15–20
- **Duration:** 5–10 seconds per GIF
- **Format:** GIF (under 5MB each for GitHub rendering)
